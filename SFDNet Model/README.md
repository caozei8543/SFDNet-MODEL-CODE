# SFDNet: Spatial-Frequency Dual-Domain Network for Real-World Low-Light Image Enhancement

## 📌 Overview

SFDNet is a novel spatial-frequency dual-domain network that addresses
the ill-posed problem of real-world low-light image enhancement.

### Core Components:
- **FAPM**: Frequency Amplitude-Phase Modulator
- **SCB**: Spatial Coordinate-aware Branch
- **CDMA**: Cross-Domain Mutual Attention
